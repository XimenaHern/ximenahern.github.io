Projects

Here are a few of my favorite projects I created in CodeHS.

Favorite Things

For this project I created a table that lists all of my favorite things for each day of the week. It includes favorite food, activity, person, and place. It is styled to make it appealing and easy to view and understand. This was a short activity to show a little bit about me and practice styling tables.

Endangered Animals

This project lists 4 animals, it includes their scientific names, where they live, and whether or not they are endangered. The information was styled to make it appealing and easy to read. This tests my design skills and my ability to make a readable website.

Lomeli's Menu

In the project above, my task was to add CSS styles to the menu items to organize the text and make it easier to read by highlighting or changing the font color of the items. This project was simple but fun and good practice using classes and ID's to style specific sections of a page.
